# Task Management API

## Project Overview
This is a simple RESTful API for managing tasks, built with ASP.NET Core 10.0.
It supports creating, updating, and retrieving tasks with validation rules.

## How to Run the API
1. Navigate to the TaskManagementAPI folder:
   ```bash
   cd TaskManagementAPI
2. Restore packages:
``` bash
    dotnet restore
```

3. Run the API:

``` bash 
dotnet run 
```


4. Open Swagger UI to test the endpoints:

``` bash
https://localhost:5001/swagger
```

## How to Run Unit Tests

1. Navigate to the test project:

``` bash
cd TaskManagementAPI.Tests
```


2. Run tests:

``` bash 
dotnet test
```
